# 🌿 Pausa – Neck & Posture Timer

> A smart work-rest timer built for people who work long hours at a laptop. Built with vibe coding.

**[▶ Live Demo](https://shrouk-cyber.github.io/Pausa/)**

---

## What it does

Pausa tracks your work session and automatically reminds you to take posture-saving breaks based on research-backed intervals:

| Break Type | Frequency | Duration | Exercises |
|---|---|---|---|
| Micro-break | Every 25 min | 45 sec | Stand up, shoulder rolls |
| Movement Break | Every hour | 3 min | Chin tucks, chest opener, shoulder rolls |
| Long Reset | Every 90 min | 12 min | Full posture recovery + eye rest |

Each break comes with **animated step-by-step exercise guides** and a **gentle sound alert**.

## Features

- ⏱ Live work session clock with 8-hour progress bar
- 🔔 Three-tier break system with animated exercise guides
- 🎵 Gentle audio alerts (Web Audio API)
- 📋 Break history log
- 📱 Installable PWA — works offline, adds to home screen
- 💾 Break history persists in localStorage

## Tech

Built entirely in HTML/CSS/JS — no frameworks, no dependencies, no build step.

## Install as App

1. Open the live demo link on your phone
2. Tap **"Add to Home Screen"** in your browser menu
3. Pausa appears on your home screen like a native app

## Run Locally

```bash
git clone https://github.com/your-username/pausa.git
cd pausa
# open index.html in any browser
```

## Deploy to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to **main branch / root**
4. Your app is live at `https://your-username.github.io/pausa`
